Data
Adding data

Ken-Q analysis makes it easy to manually input Q-sorts, or to import data from other software.

User Input

PQMethod DAT file

Excel

Ken-Q Analysis can import Excel files in three formats

  • Type 1 - Q-sort data arranged by Q-sort value
  • Type 2 - Q-sort data arranged by statement number
  • Type 3 - Ken-Q Analysis output file

Type 1 - This format is the best choice when importing hand-written Q-sort information. The recorded statement numbers should be recorded in the appropriate Q-sort value location.
